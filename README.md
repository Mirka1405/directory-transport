# directory-transport
ENG
Move files around directories.
Version which is *supposed* to work with unicode is in "unicode" branch
Example config file:
```
//print all info to screen
DEBUG
//set delay to 5 seconds
DELAY 5
//Both local and global paths are allowed
A > B
C:/Temp > C:/Users
```
